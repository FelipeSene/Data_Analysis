# Análise de Dados - E-commerce de Hardware (SQL)

Este projeto simula o banco de dados de uma loja de hardware, desenvolvido para aplicar conceitos de SQL voltados à análise estratégica e geração de insights.

## Habilidades Aplicadas
- **Modelagem de Dados:** Estruturação de tabelas relacionais com Chaves Primárias e Estrangeiras (`PK/FK`).
- **Consultas Avançadas:** Uso de `INNER JOIN` para consolidar dados de Vendas, Clientes e Produtos.
- **Funções de Agregação:** Cálculos de métricas de negócio utilizando `SUM()`, `AVG()` e `COUNT()`.
- **Manipulação e Filtros:** Segmentação de dados e organização de relatórios com `WHERE`, `GROUP BY` e `ORDER BY`.
- **Filtros Temporais Dinâmicos:** Uso de `CURRENT_DATE` e `INTERVAL` para criação de relatórios de vendas recentes (últimos 30 dias).
- **Consultas Avançadas:** Uso de `JOIN` para consolidar dados de múltiplas tabelas.

## Insights de Negócio Extraídos
- **Performance de Vendas:** Identificação das marcas com maior participação no faturamento total.
- **Giro de Estoque:** Monitoramento de produtos de alta performance (DDR5) com baixa disponibilidade.
- **Relatório de Vendas Recentes:** Filtragem de transações do último mês para monitoramento de metas e KPIs mensais.

## Tecnologias
- **MySQL** (Gerenciamento de Banco de Dados)
- **Git & GitHub** (Versionamento e Documentação Técnica)
